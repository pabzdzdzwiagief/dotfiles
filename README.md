dotfiles
========

pabżdźdźwiągief's configuration files and scripts.

Programs used:
- emacs
- nitrogen
- parcellite
- dmenu
- i3
- mpd
- urxvt
- tmux
- zsh

installation
------------

Clone this repo somewhere (e.g. in `~/.config`) and run `make install` to
symlink its contents inside the `$HOME` directory.

Installation requirements:
- GNU Make
- GNU Wget
- Python
